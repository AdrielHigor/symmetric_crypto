# symmetric_crypto

## Resumo / Summary
O objetivo deste script simples é permitir a criptografia simétrica de qualquer dado, para garantir segurança e privacidade. /
The objective of this simple script is to allow a symmetric cryptography of any data you want, to ensure security or data privacy.

### Pré-requesitos / Requirements
* **Python3 >**

### Instalação / Installing

É necessario que você possua em seu computador uma pasta com um `virtualenv` ativo em algum diretório do seu computador, caso não tenha, siga os passos a seguir: /
Make sure to have a "virtualenv" active, if not, the following steps can help you to set one:

```sh
$ python3 -m venv env
$ source env/bin/activate
```

Agora você pode clonar o repositório do `symmetric_crypto` /
Now you can clone the 'symmetric_crypto' repository.

```sh
$ git clone https://github.com/AdrielHigor/symmetric_crypto.git
```

Depois de terminar esta etapa, é preciso que o executar a instalação das bibliotecas necessárias para rodar o projeto `symmetric_crypto/requirements.txt` .Execute o seguinte comando: /
After finishing this step, you'll need to execute the needed libraries' installation to run the project `symmetric_crypto/requirements.txt`. Execute the following command:

```sh
(venv)$$ pip install -r requirements.txt
```

Ou / Or

```sh
(venv)$ python -m pip install -r requirements.txt
```

Agora com tudo pronto basta executar o `main.py` para ter uma interface gráfica, ou se preferir, criar sua própria aproximação usando as classes presentes em `sym_crypto.py`. /
Now with everything set you only need to execute the `main.py` so you can see the GUI, or if you prefer, create your own aproach using the classes in `sym_crypto.py`.
